# Game Data Fill — Progress Tracker

Incremental authoring of accurate per-character/weapon/buff data for the two
game modules. **All multiplier tables & buff values are best-effort from
knowledge and should be verified against the game / a data source.** This file
tracks what's been filled so we can continue in batches.

## Where the data lives
- **Character skills** (per-skill scaling + 10-level talent table): `adapters/game-definitions/<game>/skills.ts` → `CHARACTER_SKILLS[<charId>]`
- **Kit buffs** (team buffs a character deploys, used by Party Setup): `<game>/bundle.ts` → `supplements.buffs.character` (matched to a member by the character's display **name**)
- **Weapons** (base ATK + secondary; optional `buffs` for team-buff support weapons): `<game>/weapons.ts`
- **Set bonuses**: already derived from the module's real `def.sets` — no manual work needed
- **Base character stats / element / weapon / rarity**: `<game>/characters.ts` (full roster already populated)

## Per-character checklist legend
`S` = precise skills · `K` = kit team-buff · `—` = n/a (character has no team buff) · `·` = not done yet
